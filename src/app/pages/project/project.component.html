Sistema de Facturación

API REST completa para gestión de facturas, clientes y inmuebles

Descripción General

Sistema completo de facturación empresarial que permite gestionar clientes, inmuebles, crear facturas y generar reportes. Desarrollado con Node.js, Express y MySQL siguiendo arquitectura REST.

Tecnologías utilizadas:

Node.js Express MySQL JWT Auth PDF Generator

Funcionalidades Principales

👥 Gestión de Clientes

GET
/api/clients

Obtiene la lista completa de clientes registrados.

POST
/api/clients

Registra un nuevo cliente con datos fiscales completos.

📦 Gestión de Inmuebles

GET
/api/states

Lista todos los productos con precios y stock disponible.

PUT
/api/states/:id

Actualiza información y de inmuebles existentes.

🧾 Sistema de Facturación

POST
/api/invoices

Crea una nueva factura con cálculo automático de IVA y totales.

GET
/api/invoices/:id/pdf

Genera y descarga la factura en formato PDF.

📊 Reportes y Analytics

GET
/api/reports/sales

Genera reportes de ventas por períodos personalizables.

📋 Características Técnicas

  • Base de datos MySQL con relaciones optimizadas entre tablas.
  • Autenticación JWT para proteger endpoints administrativos.
  • Generación automática de PDFs con formato profesional.
  • Cálculo automático de IVA y descuentos por productos.
  • Validación completa de datos fiscales y comerciales.
  • Sistema de numeración automática de facturas.
