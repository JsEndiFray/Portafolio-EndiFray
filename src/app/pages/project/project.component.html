Documentación de la API

Esta API permite gestionar usuarios, roles y autenticación de manera eficiente. A continuación, un breve resumen de los principales endpoints:

1. Autenticación

POST /auth/login: Autentica un usuario y devuelve un token JWT para acceder a los recursos protegidos.

2. Gestión de Usuarios

GET /todosgi : Recupera la lista de usuarios registrados (requiere autenticación).

POST /admin: Permite crear un nuevo usuario proporcionando datos como nombre, correo y rol.

3. Gestión de Roles

GET /roles: Recupera la lista de roles disponibles (requiere autenticación).

Notas importantes

  • Las rutas protegidas utilizan el decorador @Roles para controlar el acceso.
  • Las rutas públicas, como la autenticación, están marcadas con el decorador @Public.
